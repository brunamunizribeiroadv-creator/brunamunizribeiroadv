BRUNA MUNIZ RIBEIRO - SITE INSTITUCIONAL

1️⃣ CRIE O REPOSITÓRIO NO GITHUB:
   Nome: brunamunizribeiroadv

2️⃣ NO SEU COMPUTADOR:
   git clone https://github.com/brunamunizribeiro/brunamunizribeiroadv
   cd brunamunizribeiroadv

3️⃣ COPIE OS ARQUIVOS DESTE ZIP PARA ESSA PASTA

4️⃣ ENVIE PARA O GITHUB:
   git add .
   git commit -m "Publicação inicial do site"
   git push origin main

5️⃣ ATIVE O GITHUB PAGES:
   - Vá em Settings > Pages
   - Em "Branch", selecione main e "/" (root)
   - Clique em Save

O site ficará disponível em:
https://brunamunizribeiro.github.io/brunamunizribeiroadv
